## 🚀 新增功能：玩家卡片盒 (Inventory System)
* **動態儲存**：使用 JavaScript `Array.push()` 實作資料持久化，抽到的卡片會存入玩家背包。
* **即時渲染 (Live Rendering)**：每當資料陣列更新，系統會自動遍歷 (Traverse) 陣列並重新繪製 UI。
* **視覺分層**：區分「本次抽卡結果（大圖）」與「卡片盒（縮圖）」，提供更直覺的 UX 體驗。
